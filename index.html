Manifetch

A Unified Web-Based Platform for Real-Time NICU Signal Monitoring

İrem Onaran  ·  Merve İçkilli  ·  Sacide Sena Filiz  ·  Şevval Neva Varol

About

Manifetch is a secure, web-based clinical decision support platform for Neonatal Intensive Care Units. It monitors simulated neonatal physiological signals in real time and generates AI-assisted risk alerts for healthcare staff — within a single-hospital, on-premises environment.

Key Features

Real-Time MonitoringECG, SpO₂, Heart Rate, Respiration Rate via WebSocket
AI Risk ScoringMulti-label inference: sepsis, apnea, cardiac anomaly
SHAP ExplainabilityTop-3 contributing features per prediction
Dual-Layer AlertingRule engine + AI, ACTIVE → ACKNOWLEDGED → RESOLVED
Role-Based AccessDoctor, Nurse, Administrator with JWT
PDF ReportsClinical report export with TR/EN support

Tech Stack

FastAPI PostgreSQL React WebSocket scikit-learn LightGBM XGBoost SHAP NeuroKit2 AES-256-GCM
